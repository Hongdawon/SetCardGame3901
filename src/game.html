Set Card Game

Set Card Game Instructions

This game consists of three 60 second rounds for each player.
The goal each round is to select as many 3-card "Sets" as possible.
A "Set" consists of 3 cards where each feature (color, shape, number, and shading) is either the same on all cards or different on each.
If a valid set is found, you earn points. If not, try again!
And finally, the Hint button can only be used once.
